
1) This module is a maven project and will run on JRE 8, and it is creating in memory data, so no database connection required. It has a main class Appication.java which will initiate spring boot and avail required service.
2) Hit http://localhost:8080/s/YYC/YYZ to search the flights
3) On search, list size check has been placed to give proper message of no flights availability on search.
4) Sorting first will happen on price and then will sort on departure dates.
